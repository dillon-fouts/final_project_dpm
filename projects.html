College Majors Database Project

In collaboration with Megan Mullarkey, Tyler Noga, Allison Hersom, and Lucas Hayes, I contributed to the creation of a web-based database application aimed at guiding individuals through the selection of college majors based on post-graduate employment data and salary expectations.

The project leveraged a dataset from the “The Economic Guide to Picking a College Major” and included comprehensive data on 173 majors. This data encompassed employment statistics, median salaries, and other crucial metrics, transformed into a searchable, user-friendly database interface.

Through meticulous data normalization, ER diagramming, and SQL schema design, we built a platform that not only provided key insights into career prospects but also honed our skills in database management and application development.

Although the web application is not currently hosted online, the project was a pivotal experience in practical database application and user interface design.
